CSS Selectors

Selector Example Description
Element head selects all <head> elements
Class .example selects all class="example" elements
Id #example selects the id="example" element

Semantic HTML

Name Tag Description
Article <article> Represents a part of a page which is self-contained and could be published elsewhere.
Header <header> Represents a part of a page which is meant to be introductory.
Footer <footer> Represents a part of a page which is meant to be at the end of a completed block of content.
